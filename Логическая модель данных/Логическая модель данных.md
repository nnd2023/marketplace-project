# Логическая модель данных

## Общее описание
В данном разделе описана логическая модель данных: перечислены основные сущности (классы объектов), их свойства (атрибуты) и взаимосвязи (один‑ко‑многим, многие‑ко‑многим и т.д.). Модель отражает требования бизнеса и заказчика, служа основой для проектирования физической базы данных.

## Описание атрибутов
Для удобства чтения атрибуты сгруппированы по сущностям, но оформлены единой таблицей.

| Атрибут | Тип | Ограничения | Обязательность | Описание |
|:---|:---|:---|:---:|:---|
| **`UserAccount.user_id`** | Identifier | Уникальный (PK) | да | Уникальный идентификатор учетной записи |
| `UserAccount.email` | String | Уникальный, валидный формат email | да | Адрес электронной почты |
| `UserAccount.phone` | String | Формат международного номера | нет | Контактный телефон |
| `UserAccount.password_hash` | String | | да | Криптографический хеш пароля |
| `UserAccount.first_name` | String | | нет | Имя пользователя |
| `UserAccount.last_name` | String | | нет | Фамилия пользователя |
| `UserAccount.status` | StatusEnum | `active`, `banned`, `pending` | да | Текущий статус учетной записи |
| `UserAccount.created_at` | DateTime | | да | Дата и время регистрации |
| `UserAccount.updated_at` | DateTime | ≥ `created_at` | нет | Дата последнего обновления профиля |
| **`UserRole.role_id`** | Identifier | Уникальный (PK) | да | Уникальный идентификатор роли |
| `UserRole.role_name` | String | Уникальный | да | Наименование роли (напр. `customer`, `seller`) |
| `UserRole.permissions` | List\<String\> | Массив строк/прав | нет | Список разрешений, назначенных роли |
| **`UserAccountRole.user_id`** | Identifier | FK → `UserAccount`, часть составного PK | да | Идентификатор пользователя |
| `UserAccountRoles.role_id` | Identifier | FK → `UserRole`, часть составного PK | да | Идентификатор роли |
| **`SellerStore.store_id`** | Identifier | Уникальный (PK) | да | Уникальный идентификатор магазина |
| `SellerStore.owner_user_id` | Identifier | FK → `UserAccount`, Уникальный | да | Идентификатор владельца (аккаунта) |
| `SellerStore.store_name` | String | Уникальный | да | Отображаемое название магазина |
| `SellerStore.description` | Text | | нет | Развернутое описание для покупателей |
| `SellerStore.legal_info` | Json | Юр. лицо/ИП, реквизиты | нет | Данные для фискализации и выплат |
| `SellerStore.status` | StatusEnum | `active`, `suspended`, `pending` | да | Статус модерации магазина |
| `SellerStore.created_at` | DateTime | | да | Дата регистрации магазина |
| **`ProductCategory.category_id`** | Identifier | Уникальный (PK) | да | Уникальный идентификатор категории |
| `ProductCategory.parent_category_id` | Identifier | FK → `ProductCategory` | нет | Идентификатор родительской категории (дерево) |
| `ProductCategory.name` | String | | да | Название категории |
| `ProductCategory.slug` | String | Уникальный, латиница/цифры | да | URL-идентификатор для маршрутизации |
| `ProductCategory.is_active` | Boolean | | да | Флаг видимости категории в каталоге |
| **`Product.product_id`** | Identifier | Уникальный (PK) | да | Уникальный идентификатор товара |
| `Product.store_id` | Identifier | FK → `SellerStore` | да | Идентификатор магазина-продавца |
| `Product.category_id` | Identifier | FK → `ProductCategory` | да | Идентификатор основной категории |
| `Product.sku` | String | Уникальный | да | Артикул (Stock Keeping Unit) |
| `Product.title` | String | | да | Наименование товара |
| `Product.description` | Text | | нет | Подробное описание, характеристики |
| `Product.price` | Decimal | > 0 | да | Текущая цена за единицу |
| `Product.currency` | String | Код ISO 4217 (напр. `RUB`) | да | Валюта цены |
| `Product.stock_quantity` | Integer | ≥ 0 | да | Доступный остаток на складе |
| `Product.status` | StatusEnum | `draft`, `active`, `archived`, `moderation` | да | Статус карточки товара |
| `Product.created_at` | DateTime | | да | Дата добавления товара |
| `Product.updated_at` | DateTime | | нет | Дата последнего изменения карточки |
| **`MediaAsset.media_id`** | Identifier | Уникальный (PK) | да | Уникальный идентификатор медиафайла |
| `MediaAsset.product_id` | Identifier | FK → `Product` | да | Идентификатор привязанного товара |
| `MediaAsset.url` | String | Валидный URL | да | Прямая ссылка на файл в CDN |
| `MediaAsset.mime_type` | String | Напр. `image/jpeg`, `video/mp4` | нет | MIME-тип содержимого |
| `MediaAsset.alt_text` | String | | нет | Альтернативный текст (SEO/доступность) |
| `MediaAsset.is_primary` | Boolean | | да | Флаг главного изображения товара |
| `MediaAsset.sort_order` | Integer | ≥ 0 | нет | Порядковый номер отображения в галерее |
| **`Order.order_id`** | Identifier | Уникальный (PK) | да | Уникальный идентификатор заказа |
| `Order.customer_id` | Identifier | FK → `UserAccount` | да | Идентификатор покупателя |
| `Order.order_number` | String | Уникальный, человеко-читаемый | да | Внешний номер заказа (напр. `#10045`) |
| `Order.status` | OrderStatusEnum | `created`, `paid`, `processing`, `shipped`, `delivered`, `cancelled`, `refunded` | да | Текущий статус жизненного цикла заказа |
| `Order.total_amount` | Decimal | ≥ 0 | да | Итоговая сумма заказа (с учетом скидок/доставки) |
| `Order.currency` | String | Код ISO 4217 | да | Валюта расчета |
| `Order.delivery_address` | Structure | Адрес, индекс, координаты | да | Адрес доставки заказа |
| `Order.created_at` | DateTime | | да | Дата и время оформления |
| `Order.updated_at` | DateTime | | нет | Дата последнего изменения статуса |
| **`OrderItem.order_item_id`** | Identifier | Уникальный (PK) | да | Уникальный идентификатор позиции |
| `OrderItem.order_id` | Identifier | FK → `Order` | да | Идентификатор родительского заказа |
| `OrderItem.product_id` | Identifier | FK → `Product` | да | Идентификатор товара |
| `OrderItem.quantity` | Integer | > 0 | да | Количество единиц товара в позиции |
| `OrderItem.unit_price` | Decimal | > 0 | да | Цена единицы на момент покупки (фиксируется) |
| `OrderItem.subtotal` | Decimal | = `quantity * unit_price` | да | Сумма по позиции |
| `OrderItem.status` | OrderItemStatusEnum | `pending`, `shipped`, `delivered`, `returned` | да | Статус исполнения конкретной позиции |
| **`Shipment.shipment_id`** | Identifier | Уникальный (PK) | да | Уникальный идентификатор отгрузки |
| `Shipment.order_item_id` | Identifier | FK → `OrderItem` | нет | Идентификатор родительсткой позиции заказа |
| `Shipment.carrier_code` | String | | да | Код/наименование службы доставки |
| `Shipment.tracking_number` | String | Уникальный | нет | Трек-номер для отслеживания |
| `Shipment.status` | Enum | `created`, `picked_up`, `in_transit`, `delivered`, `failed` | да | Статус логистического процесса |
| `Shipment.shipped_at` | DateTime | | нет | Фактическая дата отправки |
| `Shipment.delivered_at` | DateTime | | нет | Фактическая дата вручения |
| `Shipment.created_at` | DateTime | | да | Дата создания заявки на доставку |
| **`Payment.payment_id`** | Identifier | Уникальный (PK) | да | Уникальный идентификатор платежа |
| `Payment.order_id` | Identifier | FK → `Order` | да | Идентификатор оплачиваемого заказа |
| `Payment.amount` | Decimal | > 0 | да | Сумма транзакции |
| `Payment.currency` | String | Код ISO 4217 | да | Валюта платежа |
| `Payment.payment_method` | String | `card`, `wallet`, `sbp`, `cash` | да | Способ оплаты |
| `Payment.status` | PaymentStatusEnum | `pending`, `authorized`, `captured`, `failed`, `refunded` | да | Статус обработки платежа |
| `Payment.gateway_transaction_id` | String | Уникальный | нет | Внешний ID в платежном шлюзе |
| `Payment.created_at` | DateTime | | да | Дата инициации платежа |
| `Payment.processed_at` | DateTime | | нет | Дата успешного проведения/захвата |
| **`Payout.payout_id`** | Identifier | Уникальный (PK) | да | Уникальный идентификатор выплаты |
| `Payout.store_id` | Identifier | FK → `SellerStore` | да | Идентификатор магазина-получателя |
| `Payout.amount` | Decimal | > 0 | да | Общая сумма к перечислению |
| `Payout.currency` | String | Код ISO 4217 | да | Валюта выплаты |
| `Payout.status` | PayoutStatusEnum | `pending`, `processing`, `completed`, `failed`, `cancelled` | да | Статус выплаты продавцу |
| `Payout.requested_at` | DateTime | | да | Дата запроса на вывод средств |
| `Payout.completed_at` | DateTime | | нет | Дата зачисления на расчетный счет |
| `Payout.bank_transfer_ref` | String | | нет | Референс банковского перевода/фискального чека |
| **`PayoutItem.payout_id`** | Identifier | FK → `Payout`, часть составного PK | да | Идентификатор выплаты |
| `PayoutItem.payment_id` | Identifier | FK → `Payment`, часть составного PK | да | Идентификатор исходного платежа |
| `PayoutItem.commission_amount` | Decimal | ≥ 0 | да | Удержанная комиссия маркетплейса |
| `PayoutItem.net_amount` | Decimal | = `payment.amount - commission_amount` | да | Чистая сумма к зачислению из платежа |
| **`Review.review_id`** | Identifier | Уникальный (PK) | да | Уникальный идентификатор отзыва |
| `Review.author_id` | Identifier | FK → `UserAccount` | да | Идентификатор автора отзыва |
| `Review.product_id` | Identifier | FK → `Product` | нет | Идентификатор товара (оценка товара) |
| `Review.store_id` | Identifier | FK → `SellerStore` | нет | Идентификатор магазина (оценка магазина) |
| `Review.rating` | Integer | Диапазон 1..5 | да | Оценка в баллах (звезды) |
| `Review.title` | String | | нет | Заголовок отзыва |
| `Review.comment` | Text | | нет | Текстовое содержимое отзыва |
| `Review.status` | ModerationStatusEnum | `pending`, `published`, `rejected` | да | Статус модерации |
| `Review.created_at` | DateTime | | да | Дата публикации |
| **`Dispute.dispute_id`** | Identifier | Уникальный (PK) | да | Уникальный идентификатор спора |
| `Dispute.initiator_id` | Identifier | FK → `UserAccount` | да | Идентификатор инициатора |
| `Dispute.order_item_id` | Identifier | FK → `OrderItem` | да | Идентификатор спорной позиции заказа |
| `Dispute.reason` | String | | да | Краткая причина обращения |
| `Dispute.description` | Text | | нет | Детальное описание ситуации |
| `Dispute.status` | DisputeStatusEnum | `open`, `under_review`, `resolved`, `closed` | да | Статус рассмотрения |
| `Dispute.resolution` | Text | | нет | Итоговое решение/комментарий модератора |
| `Dispute.created_at` | DateTime | | да | Дата создания обращения |
| **`Notification.notification_id`** | Identifier | Уникальный (PK) | да | Уникальный идентификатор уведомления |
| `Notification.user_id` | Identifier | FK → `UserAccount` | да | Идентификатор получателя |
| `Notification.channel` | String | `email`, `sms`, `push`, `in_app` | да | Канал доставки |
| `Notification.template_code` | String | | да | Код шаблона сообщения |
| `Notification.payload` | Map\<String, Any\> | JSON-совместимая структура | нет | Динамические переменные для шаблона |
| `Notification.status` | Enum | `queued`, `sent`, `delivered`, `failed` | да | Статус доставки |
| `Notification.sent_at` | DateTime | | нет | Фактическая дата отправки |
| `Notification.created_at` | DateTime | | да | Дата генерации уведомления |
