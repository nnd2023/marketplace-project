# Артефакты технического задания для системы маркетплейс.

**Маркетплейс** — цифровая платформа, которая выступает в качестве посредника, соединяющего покупателей и продавцов для совершения коммерческих сделок.

Для разработки системы маркетплейса выделены следующие артефакты:

| **Наименование артефакта** | **Описание** | 
|------|----------------|
|Заинтересованные стороны|люди, группы или организации, которые имеют интерес к продукту, могут влиять на его ход и результат, а также сами зависят от его реализации|
|Глоссарий продукта|перечень ключевых терминов, связанных с продуктом|
|Бизнес правила|правила, которые определяют или ограничивают действия в процессах, связанных с разработкой продукта|
|Роли в системе|разграничение прав доступа и функциональных возможностей для различных пользователей продукта|
