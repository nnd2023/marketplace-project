# Функциональные требования (ФТ)

Данный документ содержит перечень функциональных требований к системе. Требования сформированы на основе [бизнес-правил (BR)](https://github.com/nnd2023/marketplace-project/blob/main/%D0%91%D0%B8%D0%B7%D0%BD%D0%B5%D1%81%20%D0%BF%D1%80%D0%B0%D0%B2%D0%B8%D0%BB%D0%B0/%D0%91%D0%B8%D0%B7%D0%BD%D0%B5%D1%81-%D0%BF%D1%80%D0%B0%D0%B2%D0%B8%D0%BB%D0%B0.md).
